# ✏️ Handwritten Digit Recognition using CNN

A deep learning project that recognizes handwritten digits (0–9) using a **Convolutional Neural Network (CNN)** trained on the **MNIST dataset**.
The system provides an interactive interface where users can **draw digits or upload images**, and the trained model predicts the digit along with confidence scores.

---

## 📌 Project Overview

Handwritten digit recognition is a classic computer vision problem.
In this project, a CNN model is trained on the MNIST dataset to accurately classify digits from **0 to 9**.

The project also includes a **web-based interface** where users can interact with the model in real time.

---

## 🚀 Features

* 🧠 CNN-based deep learning model
* ✏️ Draw digits directly on canvas
* 📤 Upload digit images
* 📊 Confidence chart for predictions
* 🎲 Load random test samples
* 🧹 Clear input functionality
* 🌐 Interactive UI built with Gradio

## 🖥️ User Interface

The project includes an interactive UI where users can:

* Upload a handwritten digit image
* Draw a digit on a canvas
* Predict the digit using the trained CNN model
* View prediction confidence

---

## 📊 Dataset

This project uses the **MNIST dataset**, which contains:

* 60,000 training images
* 10,000 testing images
* 28 × 28 grayscale handwritten digits
